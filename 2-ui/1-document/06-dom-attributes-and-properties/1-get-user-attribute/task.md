# Trova l'attributo

Scrivi il codice per selezionare e leggere il valore degli elementi del documento con l'attributo `data-widget-name`.

```html run
<!DOCTYPE html>
<html>
<body>

  <div data-widget-name="menu">Choose the genre</div>

  <script>
    /* il tuo codice */
  </script>
</body>
</html>
```
