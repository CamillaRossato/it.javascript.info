# Quali valori verranno mostrati dal ciclo "for"?

Per ogni ciclo scrivete quali valori verranno mostrati. Poi confrontateli con la soluzione.

I due `alert` mostreranno gli stessi valori?

1. Forma postfissa:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. Forma prefissa:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
